Getting Started

Now, before we jump into the rest of this website, we need to understand some of the basics. First, let’s talk a little about what a computer actually is (i.e. hardware and software). Next, we are going to talk about some of the basic requirements you need in order to get the most out of this website and its resources. Last but not least, I will go over a few key features on this website. Lets get started…

What is a computer? A computer is a big collection of different hardware components. These components communicate with each other internally using what is called software. In other words, the code is what runs through the hardware in order to complete a set of instructions given by the user. To learn more about computers check out this video.
